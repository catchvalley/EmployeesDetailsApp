Home Page

The site displays the given hierarchical JSON list of employees grouped by their managers in two views, a flat tabular view and a list view in default hierarchical order.

The site uses AngularJS and Angular-UI-Router to construct 3 views, transistioning between the view using the Ui-Routers states, defined in the controller.js

The page styling is done using Bootstrap.css( panels and glyphicon ) and some custom css styles.

Please note that browser security model restrictions allow this to only run in Firefox if running without hosting on web server, as this page loads additional files.
